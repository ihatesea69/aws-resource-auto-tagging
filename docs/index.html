Why AutoTag?

Built for teams that care about cost visibility and operational hygiene.

🔍

Identity Resolution

Resolves IAM users, assumed roles, root, and federated users to a human-readable owner name.

⚡

Event-Driven

CloudTrail → EventBridge → Lambda pipeline. No polling, no agents, no cron jobs.

🛡️

Error Resilient

Exponential backoff on throttling, graceful error handling, isolated per-event processing.

📦

Single Stack

One CloudFormation template deploys everything — trail, rules, Lambda, IAM, and logs.

🔌

Extensible

Add a new service in ~15 minutes. Handler, extractor, config entry, IAM permission. Done.

🧪

Well Tested

Unit tests + property-based tests with Hypothesis. Pure extractors are tested in isolation.

Cost, Scale & Maintenance

Designed to be cheap, scalable, and hands-off from day one.

~$0
Monthly Cost
Lambda free tier covers 1M requests/month. EventBridge rules are free. CloudTrail first trail is free. S3 log storage is the only marginal cost (~$0.023/GB). For most accounts, total cost is under $1/month.
∞
Scalability
Fully serverless — Lambda scales automatically to 1000 concurrent executions by default. EventBridge handles millions of events. No servers to provision, no capacity planning needed. Works the same for 10 or 10,000 resources/day.
0
Maintenance Burden
No servers, no patching, no cron jobs. CloudFormation manages all infrastructure as code. Add new services by adding a handler file and redeploying. No database, no state to manage, no drift to worry about.
